---

🔬 Current Project

☢️ Looped Operations

A Roblox-based simulation of an Advanced Gas-cooled Reactor (AGR) facility.

Looped Operations is designed to replicate real-world plant behaviour and operator workflows, including:

- Reactor startup and shutdown procedures
- Control room operation and monitoring
- System failures and emergency response
- Real-time plant state simulation
